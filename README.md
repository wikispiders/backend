# Trivia Goat Backend



## Run
```bash
docker-compose up --build
```

## Stop
```bash
docker-compose down
```

## Running with the Dart SDK

You can run the example with the [Dart SDK](https://dart.dev/get-dart)
like this:

```
$ dart run bin/server.dart
Server listening on port 8080
```
